
# Task Manager

A simple Task Manager web application built using MERN stack.

## Features

* User authentication (login/register)
* Add, update, delete tasks
* View all tasks

## Tech Stack

* React
* Node.js
* Express
* MongoDB

## Run Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
